# 🎮 Claw Wars  

**Claw Wars** es un **vertical slice** en desarrollo que se enfoca en las **mecánicas básicas**, algunas **animaciones** y comportamientos iniciales del juego.  

## 🛠 Tecnologías  
- 🎮 **Motor:** Unity 6  
- 📝 **IDE:** JetBrains Rider  

## 🚀 Estado del Desarrollo  
- 📌 **Versión actual:** Pre-Alpha `v0.1.0`
